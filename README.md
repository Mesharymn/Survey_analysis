# Survey Analyzer

A beginner-friendly Python project that analyzes survey responses from a CSV file and generates useful summaries, percentages, charts, and a text report.

## Features
- Load survey responses from CSV
- Generate survey summaries
- Calculate percentages
- Create charts automatically
- Export report files

## Run The Project

```bash
pip install -r requirements.txt
python src/main.py
```

## Tech Stack
- Python
- pandas
- matplotlib
